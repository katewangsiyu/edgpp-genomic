\paragraph{Pipeline.}
HCCP is a three-stage post-hoc framework on top of any probabilistic classifier: (1) aggregator $\phat$, (2) heteroscedastic head $\shat$, (3) Mondrian-$(y \times \shat\text{-bin})$ conformal calibration. Each stage is decoupled, so HCCP accepts any $(\phat, \shat)$ pair as a drop-in. Algorithm~\ref{alg:hccp} states the full pipeline; per-cell small-cell fallback to the pooled T2 threshold triggers when $n_{k,b} < \lceil 1/\alpha\rceil$.

\begin{algorithm}[t]
\caption{HCCP --- Heteroscedastic Class-Conditional Conformal Prediction}
\label{alg:hccp}
\begin{algorithmic}[1]
\Require Data $\D$, test chrom $c^\star$, miscoverage $\alpha$, bin count $K$
\State $\D_{\text{tr}} \gets \{i : C_i \neq c^\star\}$; fit $\phat$ on $\D_{\text{tr}}$ with chrom-LOO OOF predictions $\{\phat^{(-C_i)}(X_i)\}$
\State Fit $\shat$ on $\D_{\text{tr}}$ with targets $\{|Y_i - \phat^{(-C_i)}(X_i)|\}$ under Gaussian NLL
\State Compute $\shat$-bin edges via calibration-fold quantiles; calibration scores $s_i = |Y_i - \phat(X_i)| / (\shat(X_i) + \epsilon)$
\State Partition $\D_{\text{tr}}$ by $(Y_i, \mathrm{bin}_K(\shat(X_i)))$; compute $\qhat_{k, b}$ via Eq.~\eqref{eq:qhat}
\For{cells with $n_{k,b} < \lceil 1/\alpha\rceil$}
    \State $\qhat_{k, b} \gets \qhat_k$ \Comment{T2 pooled class-conditional fallback}
\EndFor
\For{$x \in \D_{\text{te}}$}
    \State $\pset(x) \gets \{k \in \{0, 1\} : |k - \phat(x)| \leq \qhat_{k, b(x)} \cdot \shat(x)\}$
\EndFor
\State \Return $\{(x, \pset(x))\}$
\end{algorithmic}
\end{algorithm}

\paragraph{Why the joint $(y \times \shat\text{-bin})$ partition.}
Pure class-conditional calibration (Mondrian on $y$ alone) \citep{vovk2003mondrian} fixes the imbalance pathology of split CP but leaves within-class local coverage unaddressed: our ablation reaches $\mathrm{cov}_{|Y=1} = 0.896$ on Complex but the hardest $\shat$-decile has coverage $0.454$. Stratifying by $\shat$-bin alone re-introduces minority-class pathology because $\shat$-distribution is dominated by negatives. Among Mondrian partitions over $\{y, b(x)\}$, the product partition $(y \times b)$ is the unique refinement that admits both T2 (which requires conditioning on $y$) and T3 (which requires conditioning on $b$); the two strict coarsenings each violate one of the two coverage targets.

\paragraph{$K$ selection: nested chrom-LOO.}
We select $\hat K(c_{\mathrm{outer}})$ per outer fold via \emph{proper nested chrom-LOO cross-validation}, ensuring $\hat K$ is a deterministic function of the outer calibration set alone (CP exchangeability-preserving). For each outer fold, run inner chrom-LOO and pick $\hat K(c_{\mathrm{outer}}) = \operatorname{argmin}_K \overline G_{\mathrm{inner}}(K)$ where the inner-fold worst-cell gap is measured on a fixed-$K_{\mathrm{eval}}$ sigma-bin partition (not on each candidate $K$ itself, which would artificially favour smaller $K$ via fewer cells). $K_{\mathrm{eval}}$ is set per dataset to keep $\geq 100$ minority-class samples per cell: $K_{\mathrm{eval}} = 3$ on Mendelian, $K_{\mathrm{eval}} = 5$ on Complex. Modal $\hat K(c_{\mathrm{outer}})$ is $5$ on Complex and $5\text{--}8$ on Mendelian (App.~\ref{app:tables}).
