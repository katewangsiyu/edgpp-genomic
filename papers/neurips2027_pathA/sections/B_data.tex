\subsection{TraitGym benchmark}
\label{app:data:traitgym}

We use TraitGym \citep{benegas2025traitgym}, a standardized evaluation suite for variant effect prediction. Two matched datasets:
\begin{itemize}
    \item \textbf{Mendelian (matched\_9)}: 338 ClinVar pathogenic variants + 3,042 matched benign controls = 3,380 total across 113 OMIM phenotypes. Matching on consequence type, MAF, and TSS distance (9 strata).
    \item \textbf{Complex (matched\_9)}: 1,140 GWAS-significant variants + 10,260 matched controls = 11,400 total across 29 complex traits.
\end{itemize}

\paragraph{Chrom-LOO protocol.}
Test chromosomes: $\{17, 18, 19, 20, 21, 22, X\}$ (TraitGym default). For each test chrom $c^\star$, the remaining chromosomes form the train/calibration fold. Primary metric: $\mathrm{AUPRC}_{\mathrm{chrom\text{-}weighted}} = \sum_c w_c \cdot \mathrm{AUPRC}_c$ with $w_c = n_c / \sum_{c'} n_{c'}$.

\subsection{Aggregator hyperparameters}
\label{app:data:hparams}

\paragraph{Base classifier $\phat$.}
\texttt{HistGradientBoostingClassifier} (scikit-learn 1.4): \texttt{max\_depth=2}, \texttt{max\_iter=100}, \texttt{learning\_rate=0.1}, \texttt{class\_weight=balanced}, \texttt{random\_state=42}, \texttt{early\_stopping=False}. Feature imputation: mean via \texttt{SimpleImputer}. No feature scaling (GBM is scale-invariant).

\paragraph{Heteroscedastic head $\shat$.}
\texttt{HistGradientBoostingRegressor} with identical backbone parameters. Target: $r_i = |Y_i - \phat^{(-C_i)}(X_i)|$ (chrom-LOO OOF residuals). Loss: squared error (Gaussian NLL approximation). Floor: $\shat(x) = \max(\shat_{\mathrm{raw}}(x), \epsilon)$ with $\epsilon = 10^{-4}$.

\paragraph{Conformal calibration.}
Score: $s(x, y) = |y - \phat(x)| / (\shat(x) + \epsilon)$. Bin count: $K = \hat K_{\mathrm{CV}}$ (adaptive, §\ref{sec:theory:t5}); fallback $K = 5$ for fixed-K baselines. Cell minimum $n_{k,b} \geq 5$; fallback to class-pooled $\qhat_k$ otherwise. Miscoverage: $\alpha = 0.10$.

\paragraph{DEGU-lite ensemble.}
$M = 10$ GBM members, each with bootstrap resampling (resample with replacement) and unique \texttt{random\_state}. Ensemble uncertainty: $\shat_{\mathrm{DEGU}}(x) = \mathrm{std}_{m=1}^M \phat_m(x)$. Same backbone as aggregator.

\subsection{Within-cell KS audit}
\label{app:data:ks}

To assess A2-cell (score stationarity within Mondrian cells across chromosomes), we ran 155 two-sample KS tests (5 bins $\times$ 2 classes $\times$ up to 22 calibration chroms, restricted to cells with $n \geq 20$). Results: 12.3\% rejection at $p < 0.05$ (expected 5\% under null); max KS statistic 0.24, mean 0.06. Conclusion: A2-cell is approximately but not exactly satisfied; T3$'$ (robust form) absorbs the residual via the chrom-TV term.

\subsection{Compute}
\label{app:data:compute}

All experiments on a single machine with 8$\times$ NVIDIA T4 GPUs (SM 7.5, no BF16). The HCCP pipeline (aggregator + $\shat$ head + conformal calibration) runs entirely on CPU via scikit-learn; a full chrom-LOO sweep takes $<$4 CPU-hours per feature set. DEGU-lite ensemble ($M=10$) takes $\approx$35 CPU-minutes (Mendelian) and $\approx$75 CPU-minutes (Complex). Figure and proof generation: negligible.
